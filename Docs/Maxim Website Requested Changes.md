# Maxim Website Requested Changes

Website-only notes from the Adam x Andrew meeting. CRM items are intentionally excluded except where Adam wanted the public website to support subcontractor bid/document access.

## Overall Direction

- Prioritize launching the website before any CRM work.
- Move away from a single-page brochure layout and build a small multi-page website.
- Target roughly four to five pages:
  - Home / landing page
  - Services page
  - Process page
  - About page
  - View Blueprints / bid documents page
- Keep the top navigation concept: Projects, Services, Process, About.
- Keep the main "Get Quote" call to action.
- Use the website to show authority and trust for industrial plants, commercial clients, and larger corporate clients.

## Home Page Changes

- Lighten the large structure background image by a couple shades so it does not feel so dark or gloomy.
- Keep the current image for now, but leave room to swap it later if better Maxim project photos are provided.
- Keep the "Building Excellence from Concept" direction. Adam liked that positioning.
- Update the main copy to remove redundant wording around "industrial."
- Preferred copy direction:

  > Executing high-stakes infrastructure, industrial, and commercial projects with surgical precision.

- Remove the ending phrase "and industrial authority" because Adam felt it repeated "industrial" and made the line feel awkward.
- "Laser precision" was discussed, but "surgical precision" was acceptable and cleaner.
- Add short homepage blurbs that link into the Services and Process pages.
- Keep a client-focused CTA such as "Start Your Project."
- Remove the technical specs section from the homepage if it repeats what will live on the Services or Process pages.

## Services Page

- Create a dedicated Services page instead of only using homepage sections.
- Homepage service cards should quick-link to the relevant section on the Services page, or at minimum link to the Services page.
- Reorder and rename the service cards:
  - Construction Management
  - Factory Maintenance
  - Office Renovation
  - Specialty Skills
- Swap the current order of Factory Maintenance and Office Renovation.
- Replace "Special Projects" with "Specialty Skills."
- Specialty Skills should act as a catch-all for work such as:
  - Machine rigging
  - Machine moving
  - Millwright work
  - Facility maintenance
  - Other specialized project work
- Each service section should include:
  - A short service description
  - Relevant project photos
  - Brief examples or blurbs of projects under that service category
- Include project highlights inside the Services page instead of creating a separate Projects page for now.
- Suggested service examples:
  - Construction Management: construction management / general contracting projects
  - Office Renovation: white box and office renovation projects
  - Factory Maintenance: plant maintenance work and year-round maintenance programs
  - Specialty Skills: millwright work, machine rigging, machine moving, and other specialized work

## About Page

- Create a dedicated About page.
- Include a quick company overview and company history.
- Add a credentials / memberships area.
- Mention memberships and credentials such as:
  - ISN
  - KOKA
  - Other relevant memberships or certifications Maxim provides
- Include careful wording around sister-company relationships:
  - Western Heights Waterjet / Laser Jet
  - Confab
- Keep sister-company language subtle because Adam said those ties can sometimes create conflicts with other companies.
- Use the relationships to support fabrication capability and broader resource access, not as the main headline.
- Include wording around local labor without making union language too prominent.
- Preferred wording direction:

  > Utilizing local skilled craftsmen.

- Related wording Adam liked:
  - Skilled tradesmen
  - Skilled craftsmen
  - Local skilled craftsmen

## Stats / Metrics Changes

- Replace "Current Load Capacity - 1400 metric tons" with a more relevant confidence-building metric.
- Replace "Completed Highrises" with "Completed Projects."
- Keep the safety incident rate metric because Adam liked being able to boast about safety.
- Reconsider "Managed Assets"; Adam was unsure about that metric.
- Possible replacement metric ideas discussed:
  - Active or year-round plant maintenance programs
  - 24/7 site support
  - Approximately 200,000 square feet of fabrication capability across Maxim and related companies

- Repurpose "View Blueprints" into a subcontractor-focused bid/document access page.
- This page is for contractors and subcontractors, not general clients.
- Client-facing project inquiries should use "Start Your Project" / "Get Quote."
- The page should support future access to:
  - Bid documents
  - Prints
  - Project specifications
  - Owner-provided requirements
  - Timelines
  - Upcoming or future bid opportunities
- Adam mentioned this could connect to SharePoint later.
- It can launch as an empty or simple placeholder page initially, with deeper document integration added later.
- Possible label alternatives:
  - View Blueprints
  - Bid Documents
  - Future Bids
  - Upcoming Projects
